## 📈 Recommended Learning Path

### Path A: SQL First (2-3 weeks)
```
Week 1:   SQL Sections 1-2 (Basic, Intermediate)
Week 2:   SQL Sections 3-5 (Advanced, Delta, Quality)
Week 3:   SQL Sections 6-7 + Challenges
```

Then move to Python exercises.

### Path B: Parallel Learning (3-4 weeks)
```
Week 1:   SQL Sections 1-2 + Python Sections 1-2
Week 2:   SQL Sections 3-4 + Python Sections 3-5
Week 3:   SQL Sections 5-7 + Python Sections 6-9
Week 4:   Challenges for both + Capstone
```

### Path C: Python First (2-3 weeks)
```
Week 1:   Python Sections 1-3 (Basics, Aggregations, Joins)
Week 2:   Python Sections 4-7 (Strings, Quality, I/O, ETL)
Week 3:   Python Sections 8-9 + Challenges
```

Then move to SQL exercises.

## 🎓 Challenge Descriptions

### SQL Challenges
1. **Complex aggregation** - Multi-level GROUP BY
2. **Multi-step transformation** - Use CTEs to break problem into steps
3. **Ranking with ties** - DENSE_RANK vs RANK
4. **Data reconciliation** - Find differences between tables
5. **Year-over-year analysis** - Compare periods

### Python Challenges
1. **Multi-level aggregation** - Nested groupBy operations
2. **Anomaly detection** - Statistical outliers
3. **Churn prediction** - Combine multiple conditions
4. **Data reconciliation** - Find differences with leftanti joins
5. **Dynamic schema handling** - Detect and handle types

## 💡 Pro Tips

### For SQL Exercises
✅ **DO:**
- Run each exercise one at a time
- Modify to test understanding
- Check EXPLAIN for query plans
- Write your own variations

❌ **DON'T:**
- Copy-paste without understanding
- Skip exercises
- Move to advanced too fast
- Ignore error messages

### For Python Exercises
✅ **DO:**
- Run each block separately
- Use .display() to see results
- Print intermediate values
- Experiment with transformations

❌ **DON'T:**
- Run entire file at once
- Skip import statements
- Ignore schema errors
- Copy without understanding
